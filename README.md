# Personal Portfolio - Nicolás Florentin

A modern, high-impact personal portfolio website designed to showcase web and mobile development projects. Built with a focus on visual excellence, performance, and responsive design.

[Live Demo](https://porfolio-nicolas-florentin.vercel.app/)

## Installation Instructions

To set up the project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/nicoflorentin/personal-website.git
    cd personal-website
    ```

2.  **Install dependencies:**
    This project uses `npm` as a package manager.
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```

## Usage Guidelines

This portfolio is a single-page application (SPA) built with React and Vite. 

- **Customization:** Most of the content is located in the `src/data` or `src/constants` directories (depending on the internal structure). 
- **Styling:** The project uses Tailwind CSS for layout and styling. Configuration can be found in `tailwind.config.js`.
- **Animations:** Animations are handled via `framer-motion`
- **Internationalization:** The project uses `i18next` for multilingual support.

### Running locally
After starting the dev server with `npm run dev`, open [http://localhost:5173](http://localhost:5173) in your browser to view the site.
